VaxiGraph



VaxiGraph is an interactive visualization of Polio Vaccination (Pol3) Data.

Data is taken from World Health Organization, found here.



Polio incidence is the number of new polio cases reported in a given country in a given year,

as opposed to prevalence, which is a total number of current polio cases.
